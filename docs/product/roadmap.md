# Market xG Roadmap

## Phase 0: Product and Architecture Foundation

Define the product philosophy, requirements, architecture, scoring model direction, testing strategy, and early architectural decisions.

## Phase 1: Scoring Engine MVP

Build the first local rule-based scoring engine with a small set of reliable inputs, deterministic calculations, and markdown report generation.

## Phase 2: Category Expansion

Add more market quality categories, broaden indicators, and improve score explanations while preserving explainability.

## Phase 3: Historical Backtesting

Validate score behavior against historical forward returns, drawdowns, and regime outcomes.

## Phase 4: VUSA / EUR Investor Layer

Add a VUSA-focused investor view that accounts for EUR pricing, currency effects, fund structure, and European investor context.

## Phase 5: Dashboard and Automation

Create a dashboard and scheduled reporting workflow for regular monitoring.

## Phase 6: Machine Learning and Adaptive Weighting

Evaluate machine learning, adaptive weights, and more advanced calibration after the rule-based system has been validated historically.
